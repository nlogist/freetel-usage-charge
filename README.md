# freetel-usage-charge
Gets traffic usage and charge from personal page on freetel.jp website

freetel.jp のマイページをスクレイプして，データ転送量と料金を取得します。

## Requirements
次の CPAN モジュールを利用します。
- Web::Scraper
- WWW::Mechanize
- Number::Format
- DateTime

IFTTT のアカウントがあれば連携させることができます (任意)。

## Notice
FREETELとの契約解除にともない，コードの更新を終了します。
